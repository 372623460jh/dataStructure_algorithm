# ES5与ES6大方向的区别

##作用域
- es5只有全局作用域和函数作用
    - ES5 规定，函数只能在顶层作用域和函数作用域之中声明，不能在块级作用域声明。
    ```javascript
    // es5中不合法（虽不会报错，不要这样写）
    if (true) {
        function f() {}
    }
    ```
- es6有全局作用域和函数作用和块级作用域
    - ES6 引入了块级作用域，明确允许在块级作用域之中声明函数。
    ```javascript
    // 虽然es6中合法，但也不建议这样写
    if (true) {
        function f() {}
    }
    ```
